Navegadores web

Un navegador web es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web.

Permite la visualización de documentos de texto.

Ejemplos de navegadores web

  • Google Chrome: Navegador más usado
  • Firefox: Muy usado pero eclipsado por Chrome
  • Microsoft Edge: Navegador por defecto Windows
  • Opera: Navegador sencillo más no muy usado
